pmf = [];
for j in range(len(X)):       
    exp = np.exp(-1);
    power = 1**X[j];
    fact = math.factorial(X[j]);
    val = (exp * power) / fact;
    pmf.append(val);

To get same When $\lambda$ is equal to 10, We need Normal distribution with mean equal to 10. Variance of the Poisson is also equal to $\lambda$ which will be 10 in our case. As for the Normal distribution, we need to choose $\sigma$ such that $\sigma^2 = 10$. So our $\sigma$ is around 3.2.

Following the same reasoning:
$\lambda = 1 => \mu = 1,\hspace{0.2cm} \sigma = 1$
$\lambda = 3 => \mu = 3,\hspace{0.2cm} \sigma = 1.73$
$\lambda = 5 => \mu = 5,\hspace{0.2cm} \sigma = 2.23$
$\lambda = 50 => \mu = 50,\hspace{0.2cm} \sigma = 7.07$
